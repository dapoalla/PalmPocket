# PalmPocket – Personal Finance Manager

PalmPocket is a lightweight, self-hosted personal finance manager built with PHP and MySQL. Track expenses and incomes, manage wallets, set budgets and goals, record loans, generate reports, and back up your data.

## Features

- Expenses and Incomes with receipts and notes
- Wallets: cash/bank/mobile/custom with balances and transfers
- Categories hierarchy: main (parent) and subcategories, with subcategory-only selection in forms
- Budgets (monthly, per category), progress, and exceeded notifications
- Goals: Saving and Wishlist, with dashboard summaries
- Loans: Given (as expenses) and Taken (as incomes), settlement flows
- Reports: Category Spending pie, transactions list with filters and CSV export
- Dashboard: monthly totals, parent-category spending list, wishlist/loans tiles
- Backups: create and restore
- Mobile-friendly UI with collapsible sidebar

## Requirements

- PHP 8.0+
- MySQL 5.7+ or MariaDB 10.3+
- Web server (Apache/Nginx). Sample .htaccess provided for Apache
- File uploads enabled for receipts

## Quick Start (Local – XAMPP example)

1. Clone the repo into your web root (e.g., `c:/xampp2/htdocs/pocketapp`).
2. Ensure Apache and MySQL are running.
3. Visit `http://localhost/pocketapp/index.php` to launch the installer.
4. Choose Overwrite if this is your first install or when upgrading schema.
5. Create the first user and default data via the installer.
6. Log in and start using PalmPocket.

## Installation Guide (Detailed)

1. Create a database and a database user with full privileges.
2. Configure database credentials in `config.php` (generated by installer; see `config.sample.php` for reference).
3. Navigate to `index.php` in your browser to run the installer.
4. On upgrades that introduce schema changes (e.g., category hierarchy, loans), select Overwrite in the installer, or apply equivalent SQL migrations manually if you need to preserve data.
5. After installation, log in with the credentials you created.

## Backups and Restore

- Access Backups from Settings.
- Create a backup ZIP of database export and uploaded receipts.
- Restore from a backup ZIP created by PalmPocket.

## CSV Import/Export

- Expenses: Import CSV from Expenses page. CSV must include header columns:
  - `date,wallet,category,description,amount,note`
- Sample CSV is downloadable from the Expenses page.
- Exports: Reports page provides CSV export for incomes and expenses within a date range.

## Category Hierarchy

- Define main (parent) categories and subcategories from the Categories page.
- When adding/editing expenses or incomes, only subcategories are selectable.
- Dashboard and Reports aggregate spending by main (parent) category.

## Budgets (Monthly)

- Create budgets tied to a category and a month (date range).
- Dashboard and budgets list show progress (% spent). Notifications are recorded when exceeded.
- Reports → Budgets Analytics summarizes budgets overlapping the selected period (allocated vs spent, % and exceeded flag).

## Goals and Wishlist

- Saving Goals: set target and track saved amount.
- Wishlist: track desired items with price and see total on dashboard.

## Loans

- Loans Given: mark an expense as loan_given; settle to convert to income under Debts category.
- Loans Taken: mark an income as loan_taken; settle to convert to expense under Debts category.
- Dashboard shows outstanding totals and quick links.

## Security

- Password hashing, CSRF protection, and route guarding.
- File upload validation for receipts.

## Theming & UI

- Light/Dark theme toggle in Settings.
- Responsive layout with collapsible sidebar on mobile.

## Development

- Stack: PHP 8+, MySQL, Bootstrap 5, Tailwind (CDN), Chart.js (CDN)
- Routing via a front controller (`index.php`).
- MVC-like structure: `app/controllers`, `app/views`, `app/core`.
- Installer and schema under `installer/`.
